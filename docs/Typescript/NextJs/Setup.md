# Next JS Setup

## Create the Project
```bash
pnpm create next-app@latest twitter-clone-ui --yes
```

> **NOTE**: 
> ```--yes``` skips prompts and uses the saved preferences or defaults. The default setup enables TypeScript, Tailwind CSS, ESLint, App Router, and Turbopack, with import alias @/*, and includes AGENTS.md (with a CLAUDE.md that references it).
