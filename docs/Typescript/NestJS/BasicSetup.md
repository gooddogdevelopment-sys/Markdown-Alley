# NestJS Setup
This will give the high level steps of creating a new NestJS project. More information can be found on the official site. [NestJS](https://docs.nestjs.com/first-steps)

## Prerequisite
Install [NodeJS](https://nodejs.org/en)

## NestJS CLI
NestJS provides a CLI that can be installed with npm

```bash
npm i -g @nestjs/cli
```

A new project can be create with the following:

```terminal
nest new <project-name>
```

> **Note**
> You can pass in the ```--strict``` flag to turn on script mode in Typescript. More information can be found here [Typescript Docs](https://www.typescriptlang.org/tsconfig/#strict)
